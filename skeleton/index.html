是你的骨架吗？


你应该使用Skeleton，如果你开始一个较小的项目，或只是不觉得你需要所有的更大的框架的实用程序。骨架只能绘制少量的标准HTML元素，并且包含一个网格，但通常不够开始。事实上，这个网站是建立在骷髅上，并有〜200行的自定义CSS（其中一半是对接导航）。

活版印刷


类型全部设置为rem，因此字体大小和空间关系可以基于单个<html>字体大小属性进行响应大小调整。开箱即用，Skeleton永远不会更改<html>字体大小，但是如果您需要项目，则可以使用它。所有测量的仍然是10进制，虽然如此，一个<h1>与5.0rem字体大小只是手段50px。


排版被Raleway由谷歌在1.6线高度（24像素）提供服务，设定在15rem（15像素）。其他类型的基础知识，如锚，强，重点和下划线都显然包含在内。

媒体查询


Skeleton使用媒体查询来服务其可扩展网格，同时还具有查询列表，方便您跨站点设计样式。查询是首先移动的，意思是它们的目标min-width。移动优先查询是如何构建Skeleton的网格，并且是组织CSS的最佳方法。这意味着查询之外的所有样式都适用于所有设备，然后较大的设备将针对性增强。这样可以防止小型设备解析大量未使用的CSS。查询的大小是：

  • 桌面高清：1200px
  • 桌面：1000px
  • 平板电脑：750px
  • Phablet：550px
  • 手机：4000px
				
	/* 首先查询Mobile */
	
	/* 然后查询mobile */
	@media (min-width: 400px) {}
	
	/* 然后查询phablet */
	@media (min-width: 550px) {}
	
	/* 然后查询tablet */
	@media (min-width: 750px) {}
	
	/* 然后查询desktop */
	@media (min-width: 1000px) {}
	
	/* 然后查询Desktop HD */
	@media (min-width: 1200px) {}

公用事业


Skeleton有许多小的实用程序类，作为易于使用的帮助器。有时，使用实用程序类比创建一个全新的类只是为了浮动一个元素是最好的。 
				
	/* 实用工具类 */
	
	/* 使元素获取100%宽度 */
	.u-full-width {
	  width: 100%;
	  box-sizing: border-box; }
	
	/* 确保元素不在容器外运行（对列中的图像非常有用） */
	.u-max-full-width {
	  max-width: 100%;
	  box-sizing: border-box; }
	
	/*浮动的方向 */
	.u-pull-right {
	  float: right; }
	.u-pull-left {
	  float: left; }
	
	/* 清除浮动*/
	.u-cf {
	  content: "";
	  display: table;
	  clear: both; }
